## Обзор языка

MiniC - это упрощенный C-подобный язык программирования, предназначенный для учебных целей. Язык сохраняет основные концепции языков семейства C, но значительно упрощен для облегчения реализации компилятора.

## Лексическая структура

### Кодировка
- Исходные файлы используют кодировку UTF-8
- Поддерживаются окончания строк: Unix (`\n`) и Windows (`\r\n`)

### Лексемы
```
Program = { Token } EOF

Token = Keyword | Identifier | Literal | Operator | Delimiter
```

### Ключевые слова (18)
```
Keyword = "if" | "else" | "while" | "for" | "int" | "float" | "bool"
| "void" | "return" | "true" | "false" | "struct" | "fn" | "string" | "var"
| "switch" | "case" | "default" | "break" | "continue"
```

### Идентификаторы
```
Identifier  = Letter { Letter | Digit | "_" }
Letter      = "A" | "B" | ... | "Z" | "a" | ... | "z"
Digit       = "0" | "1" | ... | "9"
```

**Ограничения:**
- Максимальная длина: 255 символов
- Чувствительность к регистру: да
- Не могут совпадать с ключевыми словами
- Допустимые символы: ASCII буквы, цифры, подчеркивание
- Первый символ: буква или подчеркивание

**Примеры:**
```
x
_x
variable1
MAX_VALUE
is_valid
counter_123
```

### Литералы

#### Целые числа
```
Integer = [ "-" ] Digit { Digit }
```

**Диапазон:** `[-2³¹, 2³¹-1]` (32-битные целые со знаком)

**Примеры:**
```
0
42
-100
2147483647    // i32::MAX
-2147483648   // i32::MIN
```

#### Числа с плавающей точкой
```
Float = [ "-" ] Digit { Digit } "." Digit { Digit }
```

**Формат:** 64-битное число двойной точности (IEEE 754)

**Примеры:**
```
0.0
3.14
-2.71828
123.456
```

#### Строковые литералы
```
String = '"' { Character | EscapeSequence } '"'

EscapeSequence = "\\" ( "n" | "t" | "r" | "\\" | '"' | "'" )
Character = любой символ кроме '"', '\\', '\n', '\r'
```

**Особенности:**
- Escape-последовательности поддерживаются
- Многострочные строки не поддерживаются
- Максимальная длина: 65535 символов

**Примеры:**
```
""                    // Пустая строка
"hello"               // Простая строка
"Hello, world!"       // Строка с пробелами
"Line 1\nLine 2"      // С escape-последовательностями
"Quotes: \"text\""    // Кавычки внутри строки
```

#### Логические литералы
```
Boolean = "true" | "false"
```

## Типы данных

### Базовые типы

#### Целочисленный тип (`int`)
- 32-битное целое со знаком
- Диапазон: `[-2³¹, 2³¹-1]`
- По умолчанию: не инициализируется (нужно явно инициализировать)

```c
int x = 42;
int y = -100;
int z;  // Неинициализированная переменная (значение не определено)
```

#### Тип с плавающей точкой (`float`)
- 64-битное число двойной точности (IEEE 754)
- По умолчанию: не инициализируется

```c
float pi = 3.14159;
float temperature = -273.15;
float ratio;  // Неинициализированная переменная
```

#### Логический тип (`bool`)
- Два значения: `true` и `false`
- По умолчанию: не инициализируется

```c
bool flag = true;
bool enabled = false;
bool status;  // Неинициализированная переменная
```

#### Пустой тип (`void`)
- Используется для функций, не возвращающих значение
- Не может быть типом переменной

#### Строковый тип (`string`)
- Последовательность символов UTF-8
- Используются двойные кавычки
- Поддерживает escape-последовательности

```c
string name = "MiniC";
string path = "C:\\projects\\minic";
```

#### Выводимый тип (`var`)
- Специальное ключевое слово для автоматического вывода типа
- **Не является отдельным типом**, а указывает компилятору вывести тип из инициализатора
- **Требует наличия инициализатора**
- После вывода тип фиксируется и не может быть изменен

```c
var x = 42;          // выводится int
var y = 3.14;        // выводится float
var z = true;        // выводится bool
var s = "hello";     // выводится string
```

### Производные типы

#### Структуры (`struct`)
Пользовательские составные типы:

```c
struct Point {
    int x;
    int y;
};

struct Person {
    string name;
    int age;
    bool active;
};
```

#### Массивы
Статические массивы фиксированного размера:

```c
int arr[10];           // массив из 10 целых чисел
float matrix[3][4];    // двумерный массив
int values[] = {1, 2, 3};  // размер выводится из инициализатора
```

## Выражения

### Арифметические выражения
```c
int a = 10 + 5;      // 15
int b = 20 - 3;      // 17
int c = 6 * 7;       // 42
int d = 100 / 4;     // 25
int e = 17 % 5;      // 2

float f = 3.14 * 2.0;  // 6.28
float g = 10.0 / 3.0;  // 3.333...
```

### Выражения с инкрементами
```c
int x = 5;
int a = x++;      // a = 5, x = 6 (постфиксный)
int b = ++x;      // b = 7, x = 7 (префиксный)
int c = x-- + --x; // сложное выражение
```

### Выражения сравнения
```c
bool b1 = (5 == 5);   // true
bool b2 = (10 != 5);  // true
bool b3 = (3 < 5);    // true
bool b4 = (7 > 2);    // true
bool b5 = (4 <= 4);   // true
bool b6 = (6 >= 9);   // false
```

### Логические выражения (с короткой схемой)
```c
bool a = true && false;  // false (правый операнд не вычисляется)
bool b = true || false;  // true (правый операнд не вычисляется)
bool c = !true;          // false

// Короткая схема предотвращает деление на ноль:
bool safe = (x != 0) && (y / x > 2);  // если x==0, y/x не вычисляется
bool check = (ptr != null) || (ptr.data == 0);  // если ptr!=null, второе условие не проверяется
```

### Вызовы функций
```c

print("result =", result);
int x = foo();
```

```c
p.x = 10;
p.y = p.x + 5;
p.x++;
```

### Доступ к элементам массива
```c

```

## Операторы

### Оператор присваивания
```c
variable = expression;
arr[index] = expression;
struct.field = expression;
```

**Составные операторы присваивания:**
```c
x += y;     // x = x + y
x -= y;     // x = x - y
x *= y;     // x = x * y
x /= y;     // x = x / y
```

### Приоритет операторов

| Уровень       | Категория         | Операторы               | Ассоциативность | Пример               |
|---------------|-------------------|-------------------------|-----------------|----------------------|
| 1 (высокий)   | Первичные         | `()` `[]` `.`           | слева направо   | `(a + b) * c`        |
| 2             | Постфиксные       | `x++` `x--`             | слева направо   | `x++ + y`            |
| 3             | Унарные           | `!` `-` `+` `++x` `--x` | справа налево   | `-x`, `!flag`, `++x` |
| 4             | Мультипликативные | `*` `/` `%`             | слева направо   | `a * b / c`          |
| 5             | Аддитивные        | `+` `-`                 | слева направо   | `a + b - c`          |
| 6             | Сравнения         | `<` `<=` `>` `>=`       | слева направо   | `x < y <= z`         |
| 7             | Равенство         | `==` `!=`               | слева направо   | `a == b != c`        |
| 8             | Логическое И      | `&&`                    | слева направо   | `a && b && c`        |
| 9             | Логическое ИЛИ    | `\|\|`                  | слева направо   | `a \|\| b \|\| c`    |
| 10 (низкий)   | Присваивание      | `=` `+=` `-=` `*=` `/=` | справа налево   | `a = b = c`          |

**Примеры с инкрементами:**
```c
int x = 5;
int y = ++x + x++;  // y = 6 + 6 = 12, x = 7
int z = x-- * --x;  // сложное выражение с правильным приоритетом
```

## Управляющие конструкции

### Условный оператор `if-else`
Скобки вокруг условия опциональны.

```c
if (condition) {
    // выполняется если condition == true
}

if condition {
    // выполняется если condition == true (без скобок)
}

if (condition) {
    // выполняется если condition == true
} else {
    // выполняется если condition == false
}

if condition1 {
    // выполняется если condition1 == true
} else if condition2 {
    // выполняется если condition2 == true
} else {
    // выполняется если все условия == false
}
```

### Оператор `switch`
```c
switch (expression) {
    case value1:
        // код для value1
    case value2:
        // код для value2
    default:
        // код по умолчанию
}
```

### Цикл `while`
Скобки вокруг условия опциональны.

```c
while (condition) {
    // выполняется пока condition == true
}

while condition {
    // выполняется пока condition == true (без скобок)
}
```

### Цикл `for`
```c
for (initialization; condition; increment) {
    // тело цикла
}

// Пример с инкрементом:
for (int i = 0; i < 10; i++) {
    // выполнится 10 раз
}

// Пример с var:
for (var i = 0; i < 10; i = i + 1) {
    // выполнится 10 раз, тип i выведен как int
}
```

### Оператор `break`
Прерывает выполнение цикла или switch:
```c
while (true) {
    if (condition) {
        break;  // выход из цикла
    }
}
```

### Оператор `continue`
Переходит к следующей итерации цикла:
```c
while (i < 10) {
    i = i + 1;
    if (i % 2 == 0) {
        continue;  // пропустить четные числа
    }
    sum = sum + i;
}
```

### Инструкция `return`
Точка с запятой опциональна перед закрывающей скобкой.

```c
return 42;
return x + y;
return x + y   // перед } точка с запятой не обязательна
return;        // для void функций
```

## Синтаксис

### Полная грамматика в EBNF

```ebnf
Program         = { Declaration | PreprocessorDirective } EOF;

PreprocessorDirective = "#define" Identifier [ Value ]
                      | "#undef" Identifier
                      | "#ifdef" Identifier Block
                      | "#ifndef" Identifier Block
                      | "#endif"
                      | "#else" Block;

Declaration     = FunctionDecl | StructDecl | VarDecl;

FunctionDecl    = "fn" Identifier "(" [ ParamList ] ")" [ "->" Type ] Block;
ParamList       = Param { "," Param };
Param           = Type Identifier;

StructDecl      = "struct" Identifier "{" { FieldDecl } "}";
FieldDecl       = Type Identifier ";";

VarDecl         = Type Identifier [ "=" Expression ] [ ";" ];

Type            = BasicType | StructType | ArrayType | "var";
BasicType       = "int" | "float" | "bool" | "void" | "string";
StructType      = "struct" Identifier;
ArrayType       = Type "[" [ Expression ] "]";

Block           = "{" { Statement } "}";

Statement       = VarDecl
                | ExprStmt
                | IfStmt
                | WhileStmt
                | ForStmt
                | ReturnStmt
                | BreakStmt
                | ContinueStmt
                | SwitchStmt
                | Block
                | EmptyStmt;

ExprStmt        = Expression [ ";" ];
EmptyStmt       = ";";
BreakStmt       = "break" [ ";" ];
ContinueStmt    = "continue" [ ";" ];

IfStmt          = "if" [ "(" ] Expression [ ")" ] Statement [ "else" Statement ];
WhileStmt       = "while" [ "(" ] Expression [ ")" ] Statement;

ForStmt         = "for" "(" ( VarDecl | ExprStmt | ";" )
                        [ Expression ] ";"
                        [ Expression ] ")" Statement;

SwitchStmt      = "switch" "(" Expression ")" "{" { CaseStmt } [ DefaultStmt ] "}";
CaseStmt        = "case" Literal ":" Statement;
DefaultStmt     = "default" ":" Statement;

ReturnStmt      = "return" [ Expression ] [ ";" ];

Expression      = Assignment;
Assignment      = LogicalOr { ("=" | "+=" | "-=" | "*=" | "/=") Assignment };
LogicalOr       = LogicalAnd { "||" LogicalAnd };
LogicalAnd      = Equality { "&&" Equality };
Equality        = Comparison { ("==" | "!=") Comparison };
Comparison      = Additive { ("<" | "<=" | ">" | ">=") Additive };
Additive        = Multiplicative { ("+" | "-") Multiplicative };
Multiplicative  = Unary { ("*" | "/" | "%") Unary };
Unary           = ( "!" | "-" | "+" | "++" | "--" ) Unary
                | Postfix;
Postfix         = Primary ( "++" | "--" | "[" Expression "]" )?;
Primary         = Literal
                | Identifier
                | "(" Expression ")"
                | FunctionCall
                | StructAccess
                | ArrayAccess;

FunctionCall    = Identifier "(" [ ArgList ] ")";
ArgList         = Expression { "," Expression };

StructAccess    = Primary "." Identifier;
ArrayAccess     = Primary "[" Expression "]";

Literal         = Integer | Float | String | Boolean;
Integer         = [ "-" ] Digit { Digit };
Float           = [ "-" ] Digit { Digit } "." Digit { Digit };
String          = '"' { Character | EscapeSequence } '"';
Boolean         = "true" | "false";

Identifier      = Letter { Letter | Digit | "_" };
Digit           = "0" | "1" | ... | "9";
Letter          = "A" | ... | "Z" | "a" | ... | "z";
EscapeSequence  = "\\" ( "n" | "t" | "r" | "\\" | '"' | "'" );
```

## Семантические правила

### Правила совместимости типов

**Совместимость при присваивании:**
- `int` совместим с `int`
- `float` совместим с `float`
- `bool` совместим с `bool`
- `string` совместим с `string`
- `int` может быть неявно преобразован в `float` (расширение)
- `float` **не может** быть неявно преобразован в `int` (сужение требует явного приведения)

**Примеры:**
```c
int a = 5;
float b = a;      // OK: int -> float
int c = 3.14;     // Ошибка: float -> int не разрешено
```

### Правила для бинарных операторов

**Арифметические операции (+, -, *, /, %):**
- `int + int → int`
- `float + float → float`
- `int + float → float`

**Операции сравнения (==, !=, <, <=, >, >=):**
- `T + T → bool` (где T - числовой тип)
- `int == float → bool` (допустимо, int преобразуется в float)

**Логические операции (&&, ||):**
- `bool + bool → bool`
- Другие типы не допускаются

### Правила для унарных операторов

**Унарный минус (-):**
- `-int → int`
- `-float → float`

**Логическое НЕ (!):**
- `!bool → bool`

**Инкремент/декремент (++, --):**
- Применимы только к переменным числовых типов
- Изменяют значение переменной

### Короткая схема вычислений

Логические операторы `&&` и `||` используют короткую схему:
- `a && b`: если `a` ложно, `b` не вычисляется
- `a || b`: если `a` истинно, `b` не вычисляется

Это позволяет безопасно писать:
```c
if (ptr != null && ptr.data > 0) { ... }  // безопасно при ptr == null
if (x == 0 || 100 / x > 5) { ... }        // безопасно при x == 0
```

### Области видимости

1. **Глобальная область** - все функции и глобальные переменные
2. **Область функции** - параметры функции
3. **Область блока** - локальные переменные внутри `{ }`
4. **Область структуры** - поля структуры

**Поиск идентификаторов:**
- Сначала в текущей области видимости
- Затем во внешних областях (до глобальной)
- Имена могут быть переопределены во вложенных областях

### Проверка функций

**Сигнатура функции:**
- Количество аргументов должно совпадать с количеством параметров
- Типы аргументов должны быть совместимы с типами параметров

**Возвращаемое значение:**
- Не-void функции должны иметь `return` с выражением
- Тип возвращаемого значения должен совместим с объявленным
- Void функции могут иметь `return;` или не иметь `return`

### Проверка переменных

**Объявление:**
- Переменная не может быть объявлена дважды в одной области
- Переменная должна быть объявлена перед использованием

**Инициализация:**
- Тип инициализатора должен быть совместим с типом переменной
- Для `var` инициализатор **обязателен**

**Вывод типа для `var`:**
- Тип выводится из инициализатора
- После вывода тип фиксируется
- Последующие присваивания проверяются на совместимость с выведенным типом

**Примеры:**
```c
var x = 42;     // x: int
x = 100;        // OK
x = 3.14;       // Ошибка: int != float

var y;          // Ошибка: var требует инициализатора
```

### Проверка структур

**Объявление структуры:**
- Поля не могут дублироваться
- Поля могут быть любого типа, включая другие структуры

**Доступ к полям:**
- Доступ через оператор `.`
- Поле должно существовать в структуре

**Пример:**
```c
struct Point {
    int x;
    int y;
};

struct Point p;
p.x = 10;       // OK
p.z = 20;       // Ошибка: поле z не существует
```

### Проверка массивов

**Объявление:**
- Размер должен быть константным целочисленным выражением
- Тип элементов может быть любым

**Доступ:**
- Индекс должен быть целочисленным
- Выход за границы не проверяется (undefined behavior)

### Проверка break и continue

- `break` допустим только внутри `while`, `for` или `switch`
- `continue` допустим только внутри `while` или `for`
- Использование вне цикла вызывает ошибку компиляции

## Восстановление после ошибок

### Стратегии восстановления

#### 1. Панический режим
Пропуск токенов до следующей точки синхронизации (`,`, `;`, `}`, ключевые слова).

#### 2. Восстановление на уровне фраз
Вставка пропущенных токенов или удаление лишних:
- Пропущенная точка с запятой: вставляем и продолжаем
- Лишняя закрывающая скобка: пропускаем
- Пропущенная скобка в условии: добавляем

## Примеры программ

### Пример 1: Демонстрация инкрементов
```c
fn main() {
    int x = 5;
    
    // Постфиксные операторы
    int a = x++;  // a = 5, x = 6
    
    // Префиксные операторы
    int b = ++x;  // b = 7, x = 7
    
    // Смешанное использование
    int c = x-- + --x;  // c = 7 + 5 = 12, x = 5
    
    return x;
}
```

### Пример 2: Функции и условия
```c
fn factorial(int n) -> int {
    if (n <= 1) {
        return 1;
    }
    return n * factorial(n - 1);
}

fn main() {
    int result = factorial(5);
    return result;
}
```

### Пример 3: Работа со структурами
```c
struct Point {
    int x;
    int y;
};

fn main() {
    struct Point p;
    p.x = 10;
    p.y = 20;
    p.x++;  // инкремент поля структуры
    
    return p.x + p.y;
}
```

### Пример 4: Вывод типов (var)
```c
fn main() {
    var x = 42;          // выводится int
    var y = 3.14;        // выводится float
    var z = true;        // выводится bool
    var s = "hello";     // выводится string
    
    x = 100;             // OK
    y = 2.71;            // OK
    
    // x = 3.14;         // Ошибка: int != float
    // var w;            // Ошибка: var требует инициализатора

}
```

### Пример 5: Сложные выражения с инкрементами
```c
fn compute(int x, int y, int z) -> int {
    int a = (x++ + y--) * (--z + ++x);
    int b = !(x > y) && (z == 0);
    return a + b;
}
```

### Пример 6: Рекурсия с var
```c
fn factorial(int n) -> int {
    if (n <= 1) {
        return 1;
    }
    var result = n * factorial(n - 1);
    return result;
}

fn main() -> int {
    var result = factorial(5);
    return result;
}
```

### Пример 7: Break и Continue
```c
fn main() -> int {
    int i = 0;
    int sum = 0;
    while (true) {
        i = i + 1;
        if (i > 10) {
            break;           // выход из цикла
        }
        if (i % 2 == 0) {
            continue;        // пропуск четных чисел
        }
        sum = sum + i;       // сумма нечетных: 1+3+5+7+9 = 25
    }
    return sum;
}
```

### Пример 8: Switch
```c
fn main() -> int {
    int x = 2;
    int result = 0;
    switch (x) {
        case 1:
            result = 10;
        case 2:
            result = 20;
        case 3:
            result = 30;
        default:
            result = 0;
    }
    return result;  // возвращает 20
}
```

### Пример 9: Короткая схема
```c
fn main() -> int {
    int a = 0;
    int b = 5;
    // Короткая схема предотвращает деление на ноль:
    if (a != 0 && b / a > 2) {
        return 1;
    }
    return 0;
}
```

### Пример 10: Массивы
```c
fn main() -> int {
    int arr[3];
    arr[0] = 10;
    arr[1] = 20;
    arr[2] = 30;
    return arr[1];  // возвращает 20
}
```
